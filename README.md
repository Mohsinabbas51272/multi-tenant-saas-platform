# Multi-Tenant SaaS Platform

This is a monorepo containing both the frontend and backend for the Multi-Tenant SaaS Platform, configured for deployment on Vercel.

## Project Structure

- `frontend/`: React + Vite application.
- `backend/`: Express.js + Prisma server.

## Deployment

Deployed on Vercel using a single repository setup with `vercel.json` routing.
